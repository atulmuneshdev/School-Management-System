School Management System

A modern School Management System built with React.js, Node.js,. This platform provides separate dashboards for Admin, Teachers, and Students to manage academic activities efficiently.

The system allows administrators to manage students, teachers, academic records, results, and school operations through a centralized and secure interface.


 Features
👨‍💼 Admin Panel

Admin dashboard overview

Manage teachers

Manage students

Academic result ledger

Class directories (1st – 12th)

Section management (A, B, C, D)

Export results as CSV

Session management

Secure login system

👩‍🏫 Teacher Portal

Manage assigned classes

Guide students academically

Upload marks

View student progress

Class management tools

🎓 Student Portal

Personal dashboard

View attendance

Check academic results

View syllabus

Assignment tracking

Timetable access

Notice board updates

Progress tracking
